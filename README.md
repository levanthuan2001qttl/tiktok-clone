### How to run this project?

#### 1. Clone project

```bash
$ git clone https://github.com/levanthuan2001qttl/tiktok-clone
```

#### 2. Change directory to folder frontend

```bash
$ cd tiktok-clone
```

#### 3. Install dependencies

```bash
$ npm install
```

#### 4. Run in developement mode

```bash
$ npm start
```
